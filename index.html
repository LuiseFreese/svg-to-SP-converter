SVG to JSON for SharePoint List Converter

This tool takes your raw SVG code and converts it into a JSON format that can be easily used in SharePoint lists for custom column formatting. Just paste your SVG code, click the button, and download the JSON file that's ready to paste into your SharePoint List.
